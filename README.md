## 🧠 What I Learned

### 📊 Excel & Data Fundamentals
- Data cleaning and structuring
This module focuses on preparing raw data so it can be used for analysis. I learned how to remove duplicates, fix formatting issues, and handle missing values. I also covered organizing data into structured tables. This made me understand that clean data is the foundation of all good analysis.

- Pivot tables and formulas
I learned how to summarize and analyze large datasets quickly. Pivot tables help me group and aggregate data without complex coding. I also useD key formulas like VLOOKUP, IF, SUMIFS, and INDEX-MATCH. These tools make Excel a powerful analysis platform.

- Data validation and formatting
This topic taught me how to control data entry and improve accuracy. I created drop-down lists, set input rules, and prevent errors. Conditional formatting helps highlight trends and issues visually. Together, these techniques improve data quality and readability.

### 📈 Power BI & Tableau
- Building interactive dashboards
I learned how to turn data into interactive visual reports. This includes adding filters, slicers, drill-through features, and dynamic visuals. Dashboards allow users to explore insights without technical knowledge. The goal is to create intuitive and impactful business intelligence solutions.

- Data Modelling (Power BI Focus)
This module teaches how to structure data properly inside Power BI. I learned about relationships, star schema design, and fact vs dimension tables. It also introduces DAX for calculated columns and measures. Strong data modelling ensures accurate, scalable, and high-performing reports

- Data storytelling techniques
This module teaches how to present insights clearly and persuasively.I learned how to guide users through a logical data narrative. The focus is on highlighting key findings rather than overwhelming viewers. Strong storytelling improves decision-making impact.

- Visual best practices
Here I understand how to choose the right charts for different data types. I avoided clutter and misleading visuals. The module covers layout design, color usage, and readability. Good design ensures insights are clear and trustworthy.

### 🗃️ SQL & Databases
- Writing SELECT queries
This topic introduces the foundation of querying databases. I learned how to retrieve data from tables using SELECT statements. It covers selecting specific columns and retrieving all records. This is the starting point for all database analysis.

- Filtering, grouping, and joins
I learned how to filter data using WHERE clauses and sort results. Grouping functions like COUNT and SUM help summarize data. Joins allow you to combine data from multiple tables. These skills are essential for real-world reporting.

- Understanding relational database structures
This module explains how databases are organized into tables and relationships. I learned about primary keys and foreign keys. It also covers normalization and data integrity concepts. Understanding structure helps you write better and more efficient queries.

### ☁️ Azure Fundamentals
- Cloud data concepts
This section introduces the basics of cloud computing and data storage. I learned how cloud platforms differ from on-premise systems. It covers scalability, availability, and cost efficiency. These concepts are key to modern data architecture.

- Introduction to Azure services
I explored core Azure services used in data projects. This includes storage, databases, and analytics tools. The module explains when and why to use each service. It provides a foundation for working in the Azure ecosystem.

- Data in the cloud ecosystem
This topic explains how data flows across cloud services. I learned learn about data ingestion, transformation, and storage in Azure. It introduces integration tools and modern data pipelines. The focus is understanding how everything connects together.

### 🐍 Python for Data
- Basic scripting
This module introduces Python syntax and core programming concepts. I learned about variables, loops, and functions. The focus is on writing simple, reusable scripts. This forms the base for data automation tasks.

- Data manipulation
I used libraries like pandas to clean and analyze data. This includes filtering datasets, transforming columns, and summarizing results. I also used libraries like MatplotLib, Seaborn, Plotly for visualizations, Numpy for mathematical operations. Python allows more advanced manipulation than Excel alone. It’s a key skill for data professionals.

- Automation fundamentals
This section teaches how to automate repetitive tasks using Python. I learned how to read files, process data, and generate outputs automatically. Automation saves time and reduces manual errors. It’s essential for scaling data workflows.
